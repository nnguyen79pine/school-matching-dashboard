EducatEd Choice

A Data-Driven School Matching System

Project Overview

EducatEd Choice helps families compare schools using data-driven insights. Instead of looking at one metric at a time, this project combines multiple academic and structural factors to create a clearer picture of school quality.

Team and Client

Team Members: Pine Nguyen, John, Victoria

Client: Dr. LaDessa Mitchell / FYG Management LLC

Datasets

  • GOSA Data
  • Georgia Insights
  • NCES

Data Processing

Data was cleaned, standardized, and combined into a master dataset. PCA was used to reduce complexity and identify key dimensions: academic performance and school stability.
